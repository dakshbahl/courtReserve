# CourtReserve

CourtReserve is a mobile app designed to make local pickleball and tennis court reservations easier, faster, and more organized.

The goal of the app is to solve a common problem at public courts: players often show up without knowing if a court is available, leading to long wait times, confusion, and unfair court usage. CourtReserve gives users a simple way to view available courts, reserve time slots, and manage their bookings directly from their phone.

## Project Overview

CourtReserve is currently focused on local public court reservations, starting with pickleball courts. The app is designed for communities where courts are shared by many players and where a clearer reservation system can improve the overall playing experience.

Instead of relying on first-come, first-served availability or informal waiting lines, CourtReserve helps organize court usage through scheduled time slots and user accounts.

## Features

* View available pickleball and tennis courts
* Select a specific court to reserve
* Book courts in 30-minute time slots
* Limit reservations to prevent court hoarding
* Manage upcoming reservations
* Cancel reservations when needed
* Clean and simple mobile interface
* Designed for local community court systems

## Planned Features

## Why I Built This

I built CourtReserve because public courts often become overcrowded, especially during peak hours. Players can waste time waiting without knowing when a court will open, and there is usually no simple system for organizing reservations fairly.

CourtReserve is meant to create a smoother experience for both players and local communities by making court availability easier to see and reservations easier to manage.

## Current Status

CourtReserve is currently in development. The first version is focused on building the core reservation flow, court selection system, and mobile user interface.

## Future Vision

The long-term vision for CourtReserve is to become more than just a booking tool. Future versions could include social features, local player communities, and partnerships with nearby businesses to reward players for using local courts and staying active.
